## 📝 Observações Importantes

### Periodicidade

- **Todos os tipos** têm periodicidade = **0** (sob demanda)
- Não há manutenções preventivas programadas automaticamente
- As manutenções devem ser agendadas manualmente conforme necessidade

### Códigos

**Padrões de Código:**
- `TM###` - Tipos básicos de manutenção (TM001-TM005)
- `BM###` - Manutenção de bomba
- `FL###` - Filtros
- `AQ###` - Aquecedor
- `SRV-###` - Serviços específicos (SRV-001 a SRV-026)

### Prioridades

- **Baixa:** Limpezas básicas, deslocamentos
- **Média:** Maioria dos serviços (59%)
- **Alta:** Reparos estruturais, trocas de filtro
- **Urgente:** Nenhum tipo definido como urgente por padrão

## 🔍 Como Usar no Sistema

### Criar Manutenção

1. Acesse **Manutenções** → **Nova Manutenção**
2. Selecione o **Equipamento**
3. Escolha o **Tipo de Manutenção** da lista (39 opções)
4. O tempo estimado e prioridade são preenchidos automaticamente
5. Ajuste conforme necessário

### Filtrar por Categoria

```sql
-- Listar apenas limpezas
SELECT * FROM tipos_manutencao WHERE categoria = 'limpeza';

-- Listar apenas para bebedouros
SELECT * FROM tipos_manutencao WHERE tipo_equipamento IN ('bebedouro', 'ambos');

-- Listar prioridade alta
SELECT * FROM tipos_manutencao WHERE prioridade_default = 'alta';
```
